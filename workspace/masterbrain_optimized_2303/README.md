## Installation

To install MasterBrain Optimized 2303, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/dre-company/masterbrain_optimized_2303.git
   ```

2. Navigate to the project directory:
   ```sh
   cd masterbrain_optimized_2303
   ```

3. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```

4. Set up environment variables in a `.env` file based on `sample.env`.
