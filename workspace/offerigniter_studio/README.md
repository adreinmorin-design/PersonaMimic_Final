# OfferIgniter Studio

OfferIgniter Studio is a lightweight copy engine for creators, operators, and digital product sellers who need sharper offers fast.

## What It Does

- Generates headlines, hooks, CTAs, and bonus ideas from a simple brief
- Works entirely offline with Python standard library only
- Produces clean text or JSON output you can reuse in landing pages, checkout copy, email, and DMs

## Quick Start

```bash
python main.py --niche "course launches" --audience "coaches" --pain-point "their pages feel flat" --outcome "more confident conversions"
```

## JSON Mode

```bash
python main.py --format json
```

## Why It Sells

People do not buy more ideas. They buy clarity, momentum, and faster execution. OfferIgniter Studio turns a vague offer into reusable sales language in one pass.
