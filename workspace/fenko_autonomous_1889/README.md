### Overview
Fenko Autonomous 1889 is a high-performance tool designed to provide elderly-friendly voice-controlled home automation for small, rural, and low-income communities. This system aims to improve the quality of life for seniors by making their homes more accessible, convenient, and safe.

### Features
- Voice-controlled interface for easy navigation
- Integration with popular smart home devices
- Customizable automation scenarios
- Real-time monitoring and alerts
- Secure and reliable data storage

### Requirements
- Python 3.8+
- Flask 2.0+
- PyTTSX3 3.4+
- SpeechRecognition 3.8+
- PyMySql 0.7+
